# Star Resonance Module Solver
This project is based on modifications to the code from [StarResonanceAutoMod](https://github.com/fudiyangjin/StarResonanceAutoMod).
And [StarResonanceAutoMod](https://github.com/fudiyangjin/StarResonanceAutoMod) originally came from [StarResonanceDamageCounter](https://github.com/dmlgzs/StarResonanceDamageCounter).

Therefore, this project continues under the GPLv3 open-source license.

## Quick Start
### Basic Usage
1. If you have not used the aforementioned plugins before, please install [Npcap](https://npcap.com/dist/npcap-1.83.exe) (included in the zip archive) or click the link to download it directly. This works with pyhton 3.12.0, i haven't tried with other version
2. Do : pip install psutil numpy opencv-python pytesseract protobuf openpyxl pandas python-docx scapy zstandard
3. Then : python solver_patched.py
4. click on Start Capture
5. Switch lane / tp or relog and your modules will be loaded

### Important Notes
The class module recommendations are based on the in-game recommendations. If you are pursuing other attributes, it is advised to use the "target attributes" and "excluded attributes" sections to find your solution.
